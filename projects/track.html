Track: Ionic application for keeping track of things

5/20/2020

If you’re like me and often buy things you don’t really use, you know the need to declutter your place every few months. I’ve gotten pretty good about doing this since moving out of my parents, but there’s still a few things that it’s hard to determine whether I really need or not like the clothes in my closet or my board games. This was the inspiration for my honors project this semester. I created a little web application I call Track that can keep track of how often I use my things and provide statistics on my usage of them.

In order to achieve this, I essentially needed a way to create a digital ‘identity’ for each of the items that I wanted to track. I figured this needed to be done in a way that was inexpensive, in case I wanted to do this on a lot of items. I liked the idea of using NFC tags, but I went with QR codes instead as they were much cheaper and I could make a bunch with a printer.

So from here the plan was as follows:

  • Create a PostgreSQL database to store our items and groups of items (I called them ‘projects’)
  • Create a web application to display and create projects and items
  • Create an API in python to make modifications the database

Then the fun part:

  • Upon item creation, generate a unique QR code that is essentially a GET endpoint to the API, which increments the count of uses on that item and updates the ‘last use date’
  • Print the QR code so I can stick it onto my hangers or board games and scan it when I use them

I didn’t run into many hiccups along the way, which is probably a first. It works nicely, and I have it hosted on a rPi server in my apartment. I look forward to seeing how much I actually use it. Here’s a few screenshots of the UI:

In the future, I’d like to extend it to be able to handle images, so I know which article of clothing goes on which hangar, but for now it’s in a pretty good place. Also, because I built it with a few projects in mind, it could probably be used for a bunch of other things. I thought it would be cool to keep track of items in your refrigerator, but the QR system would be tedious. Perhaps I could use the barcodes or utilize machine vision. Anyways, thanks for reading!
